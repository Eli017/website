While he was a freshman, Eli designed a collage using Jython.
The original photo, a single spiraling tower, was altered in
multiple ways. With the Jython code, the picture was mirrored,
shrunk, darkened, and made the photo with sepia tones.

Another project that Eli worked on was a text adventure through
Python. In this program, the user is prompted to type in answers
based upon the room they are in. The game begins by recording
the players name. Overall, this did improve skills regarding
global variables and how they are altered by the player.
